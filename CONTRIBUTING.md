# Contributing to Untouchable

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Open `Untouchable.xcodeproj` in Xcode 15+
4. Create a feature branch from `main`

## Development

- Target: macOS 13.0+
- Language: Swift 5.9+
- UI: SwiftUI with `MenuBarExtra`

## Submitting Changes

1. Keep commits focused and atomic
2. Write clear commit messages
3. Open a pull request against `main`
4. Describe what your change does and why

## Reporting Bugs

Use the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md) when filing issues.

## Code Style

- Follow standard Swift conventions
- Use `SwiftLint` defaults where applicable
- Keep files focused — one type per file when practical
